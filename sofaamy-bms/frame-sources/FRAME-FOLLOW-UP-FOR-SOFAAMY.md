# Frame Projects — Follow-up Request for Sofaamy

**Purpose:** close the remaining gaps between the current working Frame calculator and an approved production cutting list.

## The most important request

Please send **one completed Frame project as a complete evidence pack**. The project should preferably be a door or window for which the team can share:

1. Final site measurement
2. Client quotation
3. Internal material/costing sheet
4. Approved cutting list used by the factory
5. Glass schedule or glass order
6. Accessory/hardware list
7. Production or QA sheet, if available
8. A photo of the completed product

The documents must all refer to the same project and design. This will allow the system to be checked from measurement through quotation, procurement, cutting, production and installation.

## Questions to ask first: cutting-list rules

For the example project, please mark the exact pieces directly on the drawing or cutting list:

- Which profile code is used for the outer frame head, sill and jambs?
- Which profile code is used for the moving leaf or sash rails and stiles?
- Which profile code is used for an interlock, adaptor, bead, division or special member?
- How many of each piece are required for one unit?
- What is the exact cut formula from the measured width and height?
- Which pieces receive 45° cuts, 90° cuts, mitres, butt cuts, notches, holes or routing?
- What deduction or allowance is applied to each piece, and why?
- Does the formula change for single, double, or multiple-panel designs?
- Does the formula change between sliding, casement, projected, fixed, swing and hinge products?
- Are the listed dimensions finished opening sizes, frame sizes, or glass sizes?

## What we should not ask them to resend

The existing workbooks already contain much of the catalogue information: profile descriptions, codes, standard 5800 mm bars, colours, prices, accessory names and glass data. The missing information is the **measurement-to-piece rule**, not another copy of the catalogue.

## Minimum outcome for tomorrow

If the team has limited time, obtain these five answers first:

1. One real approved cutting list linked to its final measurement.
2. Exact profile code for every piece role in that list.
3. Exact cut formula, deduction and cut treatment for every piece.
4. Exact accessory quantity rule for that product.
5. Exact glass size formula and final glass dimensions.

Until these five items are confirmed, the system can generate a useful working breakdown, but the cutting list should remain marked **pending supervisor approval** rather than being treated as a final factory instruction.
